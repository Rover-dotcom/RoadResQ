# RoadResQ — Docs

This folder contains all backend API documentation and Postman files.

## Files

| File | Description |
|---|---|
| [`API_DOCUMENTATION.md`](./API_DOCUMENTATION.md) | Full API reference — all endpoints, request/response examples, pricing logic |
| [`RoadResQ_API.postman_collection.json`](./RoadResQ_API.postman_collection.json) | Postman collection — import this into Postman |
| [`RoadResQ_Local.postman_environment.json`](./RoadResQ_Local.postman_environment.json) | Postman environment — import this too |

## Postman Setup (2 steps)

1. Open Postman → **Import** → select **both** JSON files above
2. Select **"RoadResQ Local"** from the environment dropdown (top-right)

Then run the **🧪 Full Flow Test** folder in order (Steps 1–9) to test everything end-to-end.
